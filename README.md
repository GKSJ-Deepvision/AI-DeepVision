# Crowd Monitoring System using CSRNet

## Overview
Web-based crowd monitoring system using CSRNet for density estimation.
Includes real-time video inference, crowd count display, and alert logic.

## Features
- Streamlit web dashboard
- Crowd density & count estimation
- Threshold-based alert system
- SMTP email alert logic (disabled during demo for security)

## Tech Stack
Python, PyTorch, Streamlit, OpenCV

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
